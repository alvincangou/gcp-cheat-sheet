Cloud Architect Certification Cheat Sheet

Preparation Guide • Key Concepts • Decision Logic

Security & Governance

DLP API Data Loss Prevention: Mask/Redact PII.
IAP Identity Aware Proxy: Access VMs/Apps without VPN or Public IPs.
VPC SC Service Controls: Prevent exfiltration by creating a perimeter.
Binary Auth GKE: Deploy only signed/authorized images.
IAM Principle of Least Privilege: Use Custom Roles over Primitives.
CMEK/CSEK Encryption: Managed by Google, Customer, or On-prem.

Network & Load Balancing

HTTP(S) LB Global Layer 7: Best for Web apps, CDN, and WAF.
Network LB Regional Layer 4: Best for UDP/TCP traffic.
Shared VPC Centralized network for multiple projects.
Private Svc Connect Access GCP APIs via internal IP (No NAT/IGW needed).
Cloud Armor Anti-DDoS + WAF (SQLi, XSS filtering).

Storage & Data Management

Nearline Accessed < 1/mo (30-day min charge).
Coldline Accessed < 1/qtr (90-day min charge).
Bigtable High throughput, NoSQL (IoT, Time-series).
GCSFUSE Mount Cloud Storage buckets as local disks.
BigQuery Data Warehouse. Serverless. Use for SQL analytics.

Disaster Recovery (DR) Strategy

RTO (Recovery Time Objective): How long can you be down?
RPO (Recovery Point Objective): How much data loss can you tolerate?
Cold DR: Minimal cost. Recreate resources from backups/scripts (High RTO).
Warm DR: Balanced. Minimal services running (e.g., DB replication) + scale up.
Hot DR: Highest cost. Full active-active in two regions (Near zero RTO/RPO).

Compute & GKE

Istio/Mesh Handles Service-to-Service auth and visibility.
Autopilot Pay for Pods. GKE manages the nodes/security.
HPA / CA HPA = Vertical scaling (pods); CA = Cluster scaling (nodes).
Eventarc Trigger Serverless (Cloud Run) via GCP events.
