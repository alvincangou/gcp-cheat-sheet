GCP Cloud Architect Cheat Sheet

Strategic Reference for Certification & Design

Security & Governance

DLP API Mask/Redact PII in text/images.
IAP SSH/HTTPS access without Public IPs or VPN.
VPC SC Data exfiltration protection (Perimeter).
Binary Auth GKE: Only deploy trusted/signed images.
IAM Folders for Prod/Dev; Custom Roles for Least Privilege.

Messaging & ETL Logic

ServicePrimary GoalDifferentiator
Pub/SubDecouplingStreaming, 1-to-Many
Cloud TasksOrchestrationRate-limiting, scheduling
DataflowTransformationETL, Windowing, Late data
Keyword Check: "Transform" = Dataflow. "Buffer" = Pub/Sub. "Handover/Retry" = Tasks.

Networking & LBs

Global HTTP(S) LB Layer 7, Global, CDN & Cloud Armor.
Network LB Layer 4, Regional, UDP/TCP.
Shared VPC Network admin in Host, resources in Service projects.
VPC Peering Connect networks; non-transitive.

Storage & Migration

Near/Cold/Archive 30 / 90 / 365 day min storage duration.
Bigtable High-throughput NoSQL (IoT/Time-series).
GCSFUSE Mount GCS bucket to Linux VM.
DMS Serverless SQL migration (Postgres/MySQL).

Disaster Recovery (DR)

Cold: Static site/Backups. High RTO (hours/days).
Warm: (Pilot Light) DB running, App scaled to zero. Medium RTO.
Hot: Active-Active multi-region. Zero RPO/RTO. High Cost.
Exam Tip: Spanner is natively multi-regional/High Availability.

Compute & GKE

Autopilot Managed nodes; Pay-per-Pod.
Istio (Mesh) Secure pod-to-pod comms & traffic splitting.
HPA vs CA HPA = scale pods; CA = scale underlying VMs.
Eventarc Convert any GCP action into a Cloud Run trigger.

Strategic Exam Traps

Svc Transfer Svc: Cloud-to-Cloud (e.g., S3 to GCS) or Online On-prem to GCS.
Transfer Appliance: Physical box for 10TB-Petabytes of offline data.
Organization Policy: Restricts what can be done (e.g., "No external IPs"). IAM defines who.
